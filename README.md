## Animated Slider

Welcome to the Animated Slider project! This web application features a captivating image slider designed to enhance user engagement through smooth animations and transitions. Built with HTML, CSS, and JavaScript, this project provides an interactive way to showcase images or content on your website.

[Video.webm](https://github.com/user-attachments/assets/018a040b-678f-4734-b9cb-c5cb390487d8)

## Features
- Smooth Animations: Eye-catching transitions between slides that enhance the visual experience.
- Responsive Design: Optimized for various devices, ensuring a seamless experience on desktops, tablets, and smartphones.
- User Interaction: Easy navigation with buttons and indicators to control the slider.
- Customizable: Simple structure allowing easy modifications to adapt the slider to your own needs.

![Full-Screen-Capture](https://github.com/user-attachments/assets/70c6fb8d-6f29-4e33-bda9-363b4d9eaa06)

## Technology Stack
- HTML: For structuring the content of the slider.
- CSS: For styling and creating animations.
- JavaScript: To handle the logic of the slider and manage transitions.
